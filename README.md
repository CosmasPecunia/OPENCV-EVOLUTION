Détection d’Objets en Temps Réel avec YOLOv8

Projet de détection d’objets en temps réel utilisant YOLOv8, OpenCV et la camera en Python.

Ce programme capture le flux vidéo de la webcam, applique le modèle YOLO (yolov8n.pt) pour détecter les objets, puis affiche les résultats avec des rectangles, les noms des objets et le pourcentage de confiance.
